## Tech stack

- Frontend
  - React
  - React Router
  - [Shoto UI](https://shotoui.netlify.app)
- Backend
  - Express
  - Mongoose
  - MongoDB
  - JWT

## Features

- User authentication : Login/SignUp
- Wishlist : Add/Remove products
- Cart : Add/Remove products
- Product Filters : Filter by price range and customer reviews
